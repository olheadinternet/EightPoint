# 8-Point Tile Generator

A lightweight browser app for creating PlotterFiles 8-point prompt tiles as valid SVG files.

## Run

Open `index.html` in any modern browser.

## Features

- Fixed 1000×1000 tile, 1in × 1in output SVG
- Fixed stroke width of 20
- Uses all 8 endpoints with 4 paths
- Oscillation-based path shaping
- Mirror X / Mirror Y toggles
- Randomize controls and export SVG download

## Notes

The generated SVG keeps endpoints exact and produces continuous cubic Bézier paths for direct submission.
